# 🚀 File System Recovery & Optimization Tool

A simulation-based project that demonstrates how operating systems manage file storage, recover from crashes, and optimize disk performance.

---

## 📌 Overview
Modern file systems are essential for storing and managing data efficiently. However, they face issues like disk crashes, data corruption, and fragmentation.

This project simulates a file system to:
- Manage disk storage using blocks
- Handle file operations
- Simulate system behavior
- Visualize internal data organization

---

## 🧠 Features

- ✅ Virtual disk simulation  
- ✅ Bitmap-based free space management  
- ✅ File creation and deletion  
- ✅ Block allocation tracking  
- ✅ Disk structure visualization  
- ✅ Efficient storage handling  

---

## ⚙️ How It Works

- The disk is divided into **fixed-size blocks**
- A **bitmap** tracks free and occupied blocks
- Files are stored across allocated blocks
- Metadata stores file information (name, size, location)
- Visualization shows internal structure of the disk
